## 📚 О проекте

«Академия Робота-Учителя» — это интерактивная образовательная платформа, где дети изучают школьные предметы через игровую механику «Карты жизни». 

### ✨ Особенности

- 🎮 **Геймификация обучения** — проходи темы, получай звёзды и дипломы
- 🐾 **Виртуальный питомец** — заботься о питомце, играй в мини-игры
- 📐 **4 предмета** — Математика, Русский, Английский, Окружающий мир
- 🏆 **Система достижений** — зарабатывай монеты и награды
- 🌓 **Тёмная тема** — комфортное обучение в любое время
- 📱 **Адаптивный дизайн** — работает на телефонах и планшетах

### 📖 Предметы (по ФГОС 2026)

## 🚀 Быстрый старт

```bash

# Перейди в папку
cd academy-robot-teacher

# Запусти локальный сервер (например, с помощью Python)
